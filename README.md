# Retail Sales Dashboard (Power BI + Excel)

## Overview
This project analyzes retail sales data using Excel for data cleaning and Power BI for interactive dashboards.  
The dashboard provides high-level KPIs and detailed drill-down analysis by region, category, and sub-category.

## Tools Used
- Microsoft Excel (data cleaning & preparation)
- Power BI Desktop (data modeling & visualization)

## Dataset
- Sample Superstore retail sales data
- Cleaned by removing duplicates, formatting columns, and validating numeric fields

## Dashboard Pages

### 1. Sales Overview
- Total Sales KPI
- Sales distribution by Segment
- Sales by Product Category
- Sales by Ship Mode

### 2. Sales Breakdown
- Interactive slicers (Region, Category)
- Sales by Region
- Top 10 Sub-Categories by Sales
- City-level sales table with dynamic filtering

## Key Insights
- Technology is the highest revenue-generating category
- Consumer segment contributes over 50% of total sales
- Standard Class shipping dominates sales volume
- Western region leads in overall sales
